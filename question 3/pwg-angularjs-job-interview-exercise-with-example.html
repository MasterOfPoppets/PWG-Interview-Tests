AngularJS job interview exercise

Using the data-structures present in colour_groups and colours, render the following HTML select drop-down using AngularJS:

Things to note:

  • Each option's value attribute should be the ID value of the colour it holds
  • Each option's text label value should be the label value of the colour it holds
  • Where a colour has an associated colour group, it should be presented in an option group, sorted by the individual group sort ranking number (present in "sort_order" key)
  • Colours without an associated colour group should be presented at the top of the select list (without an option group)
  • The first list option should ideally have the text label of "Select a colour..." and an empty option value attribute
